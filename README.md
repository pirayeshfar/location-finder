
# یابنده هوشمند آدرس (نسخه Termux و گیت‌هاب)

این برنامه یک ابزار تک‌فایلی و سریع برای یافتن آدرس دقیق و کدپستی با استفاده از هوش مصنوعی Gemini است.

## روش اجرای سریع در Termux:

1. ابتدا پکیج‌های لازم را نصب کنید:
   ```bash
   pkg install git python
   ```
2. پروژه را دانلود کنید:
   ```bash
   git clone [آدرس-گیت‌هاب-شما]
   cd [نام-فولدر]
   ```
3. یک سرور محلی راه بیندازید:
   ```bash
   python -m http.server 8080
   ```
4. حالا در مرورگر گوشی آدرس زیر را باز کنید:
   `http://localhost:8080`

**نکته:** برای اینکه GPS کار کند، مرورگر باید به لوکیشن دسترسی داشته باشد. اگر روی سرور واقعی آپلود می‌کنید، حتماً باید پروتکل **HTTPS** فعال باشد.

## ویژگی‌ها:
- بدون نیاز به نصب کتابخانه‌های سنگین (Node_modules).
- نمایش محله، خیابان، پلاک و کد پستی ۱۰ رقمی.
- قابلیت اشتراک‌گذاری و نمایش روی نقشه گوگل.
